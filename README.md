# Generative AI Projects with LangChain

This repository contains beginner-to-intermediate Generative AI projects built using LangChain, Mistral AI, Prompt Engineering, and Python.

## Projects

### 1. Basic AI Chatbot
- Conversational AI chatbot
- Built with LangChain and Mistral AI
- Demonstrates prompt engineering and LLM integration

### 2. Movie Information Extractor
- Extracts structured movie information from paragraphs
- Generates title, genre, cast, plot, themes, ratings, and summary
- Built using LangChain Prompt Templates and Mistral AI

## Technologies Used
- Python
- LangChain
- Mistral AI
- Prompt Engineering
- Git & GitHub
